## 🔎architecture
`mvc`

## 📁 폴더 구조 
- bin : 서버 실행
- log :  로그 관리
- config : 설정 파일
- models: data 모델
- public: UI 파일
- routes: 라우팅과 컨트롤러
- views: 정적 파일

## 버전
- `v0.1.0` : 로그인과 회원가입 기능
- `v0.2.0` : 파일로 데이터 관리
- `v0.3.0` : DB로 데이터 관리
- `v0.4.0` : 로그 관리 기능 추가
